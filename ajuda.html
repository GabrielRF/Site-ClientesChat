Bot

Criar bot

  1. Fale com o @BotFather;
  2. Envie o comando /newbot;
  3. Envie o nome do bot;
  4. Envie o nome de usuário do bot. Este deve ser único e deve terminar com "bot";
  5. Caso tudo esteja correto, você receberá o TOKEN do bot.

Foto

  1. Fale com o @BotFather;
  2. Envie o comando /setuserpic;
  3. Escolha o bot que deseja colocar a foto;
  4. Envie a foto.

Modo inline

Modo disponível apenas para a equipe de atendimento.

  1. Fale com o @BotFather;
  2. Envie o comando /setinline;
  3. Escolha o bot que deseja habilitar o modo inline;
  4. Envie uma palavra ou frase que será exibida quando o bot for usado em modo inline.

Descrição

Texto exibido quando o bot é aberto pela primeira vez.

  1. Fale com o @BotFather;
  2. Envie o comando /setdescription;
  3. Escolha o bot que deseja escrever uma descrição;
  4. Envie o texto.

Sobre este bot

Texto exibido no perfil do bot.

  1. Fale com o @BotFather;
  2. Envie o comando /setabouttext;
  3. Escolha o bot que deseja escrever um texto para o perfil;
  4. Envie o texto.

Impedir grupos

Esta opção é recomendada. Deve ser feita somente após o bot ser inserido no grupo e no canal de atendimento. Ela impede que o bot seja inserido em outros grupos.

  1. Fale com o @BotFather;
  2. Envie o comando /setjoingroups;
  3. Escolha o bot que deseja impedir que seja colocado em grupos;
  4. Escolha a opção Disable.

Grupo

Criação

  1. Clique no lápis no canto inferior direito;
  2. Clique em New group.

Foto

  1. Abra o grupo;
  2. Clique na barra superior, que contém o nome do grupo;
  3. Clique no lápis;
  4. Escolha uma foto.

Adicionar bot

  1. Abra o grupo;
  2. Clique na barra superior, que contém o nome do grupo;
  3. Clique no botão azul no canto inferior;
  4. Encontre o bot e adicione-o;
  5. Defina o bot como um administrador do grupo com todas as permissões.

Adicionar pessoas

Recomenda-se que nenhuma pessoa seja adicionada ao grupo. O grupo funciona apenas para permitir os comentários no canal e para ser um registro geral de acontecimentos.

Todas as pessoas que entrarem no grupo serão removidas automaticamente pelo bot.

Certifique-se que a opção Permanecer anônimo esteja desligada para todos.

Canal

Criação

  1. Clique no lápis no canto inferior esquerdo;
  2. Clique em New channel;

Foto

  1. Abra o canal;
  2. Clique na barra superior, que contém o nome do canal;

Adicionar bot

  1. Abra o canal;
  2. Clique na barra superior, que contém o nome do grupo;
  3. Clique em Admins;
  4. Clique em Adicionar Admin;
  5. Encontre o bot e adicione-o;
  6. O bot obrigatoriamente será um administrador. Recomenda-se ligar todas as permissões.

Adicionar pessoas

  1. Abra o canal;
  2. Clique na barra superior, que contém o nome do canal;
  3. Clique em Inscritos;
  4. Clique em Convidar via Link;
  5. Compartilhe o link com todas as pessoas que farão os atendimentos.

Respostas prontas

Função disponível apenas para a equipe de atendimento.

Criação

  1. Em uma conversa em que o bot é usado, envie o comando resposta;
  2. Envie a mensagem de texto que deseja salvar como uma resposta pronta.

Uso

  1. Em qualquer chat, digite o @usuario do bot;
  2. Aguarde as opções e clique na reposta que deseja enviar.

    3. Opcionalmente é possível filtrar respostas por palavra chave, bastando digitar a palavra após o @usuario do bot.

Remoção

  1. Em uma conversa em que o bot é usado, envie o comando /remover;
  2. Envie a resposta que deseja remover.

    3. Dica: É possível selecionar a resposta usando o modo inline, facilitando assim a digitação.

Atendimento

Apenas para a equipe de atendimento.

Chat

Todos os atendimentos acontecem no canal, sendo uma conversa por usuário. A conversa de cada usuário é criada no momento em que o usuário envia sua primeira mensagem ao bot. As conversas ativas (não finalizadas) são listadas nas mensagens fixadas do canal.

A conversa com o usuário não deve ser removida do canal. Caso ocorra a remoção, em uma próxima mensagem do usuário será criada uma nova conversa, perdendo-se o histórico somente do lado da equipe de atendimento.

A remoção de mensagens é unilateral. Ou seja, caso alguém da equipe apague uma mensagem do chat, isto não terá efeito na conversa vista pela pessoa atendida. Se a pessoa atendida remover uma mensagem, esta remoção também não é refletida na conversa com a equipe de atendimento.

As mensagens podem ser editadas e a edição é efetiva tanto na conversa com o usuário quanto na conversa vista pela equipe de atendimento.

Durante uma conversa podem ser enviadas mensagens de texto, voz, vídeos, arquivos, fotos e documentos.

Prioridade

Cada conversa pode receber uma classificação de prioridade. Para ver as opções de prioridade, envie o comando /p.

A prioridade é definida quando é enviado /p e o valor da prioridade em uma mensagem só.

Exemplo: O envio de /p 3 definiria a prioridade como Urgente.

O comando /ban define a pessoa com a menor prioridade possível. A troca de mensagens ainda será possível, mas nenhuma notificação será enviada para a equipe de atendimento. Para desbanir basta definir uma nova prioridade ou finalizar o atendimento usando-se o comando /fim.

Notificação

A qualquer momento é possível solicitar que a opção de notificar quando uma conversa for iniciada. As notificações são enviadas em mensagens particulares a todas as pessoas que sejam administradoras do canal, independente de suas permissões. Ou seja, com a opção ligada, sempre que houver uma nova conversa o bot irá enviar uma mensagem em privado para cada administrador avisando que há uma conversa nova. A mensagem de notificação contém um link para fácil visualização do atendimento.
