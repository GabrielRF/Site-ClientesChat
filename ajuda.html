Bot

Criar bot

  1. Fale com o @BotFather;
  2. Envie o comando /newbot;
  3. Envie o nome do bot;
  4. Envie o nome de usuário do bot. Este deve ser único e deve terminar com "bot";
  5. Caso tudo esteja correto, você receberá o TOKEN do bot.

Foto

  1. Fale com o @BotFather;
  2. Envie o comando /setuserpic;
  3. Escolha o bot que deseja colocar a foto;
  4. Envie a foto.

Modo inline

Modo disponível apenas para a equipe de atendimento

  1. Fale com o @BotFather;
  2. Envie o comando /setinline;
  3. Escolha o bot que deseja habilitar o modo inline;
  4. Envie uma palavra ou frase que será exibida quando o bot for usado em modo inline.

Descrição

Texto exibido quando o bot é aberto pela primeira vez.

  1. Fale com o @BotFather;
  2. Envie o comando /setdescription;
  3. Escolha o bot que deseja escrever uma descrição;
  4. Envie o texto.

Sobre este bot

Texto exibido no perfil do bot.

  1. Fale com o @BotFather;
  2. Envie o comando /setabouttext;
  3. Escolha o bot que deseja escrever um texto para o perfil;
  4. Envie o texto.

Impedir grupos

Esta opção é recomendada. Deve ser feita somente após o bot ser inserido no grupo e no canal de atendimento. Ela impede que o bot seja inserido em outros grupos.

  1. Fale com o @BotFather;
  2. Envie o comando /setjoingroups;
  3. Escolha o bot que deseja impedir que seja colocado em grupos;
  4. Escolha a opção Disable.

Grupo

Criação

  1. Clique no lápis no canto inferior direito;
  2. Clique em New group.

Foto

  1. Abra o grupo;
  2. Clique na barra superior, que contém o nome do grupo;
  3. Clique no lápis;
  4. Escolha uma foto.

Adicionar bot

  1. Abra o grupo;
  2. Clique na barra superior, que contém o nome do grupo;
  3. Clique no botão azul no canto inferior;
  4. Encontre o bot e adicione-o;
  5. Defina o bot como um administrador do grupo com todas as permissões.

Adicionar pessoas

Recomenda-se que nenhuma pessoa seja adicionada ao grupo. O grupo funciona apenas para permitir os comentários no canal e para ser um registro geral de acontecimentos.

Todas as pessoas que entrarem no grupo serão removidas automaticamente pelo bot.

Certifique-se que a opção Permanecer anônimo esteja desligada para todos.

Canal

Criação

  1. Clique no lápis no canto inferior esquerdo;
  2. Clique em New channel;

Foto

  1. Abra o canal;
  2. Clique na barra superior, que contém o nome do canal;
  3. Clique no lápis;
  4. Escolha uma foto.

Adicionar bot

  1. Abra o canal;
  2. Clique na barra superior, que contém o nome do grupo;
  3. Clique em Admins;
  4. Clique em Adicionar Admin;
  5. Encontre o bot e adicione-o;
  6. O bot obrigatoriamente será um administrador. Recomenda-se ligar todas as permissões.

Adicionar pessoas

  1. Abra o canal;
  2. Clique na barra superior, que contém o nome do canal;
  3. Clique em Inscritos;
  4. Clique em Convidar via Link;
  5. Compartilhe o link com todas as pessoas que farão os atendimentos.
